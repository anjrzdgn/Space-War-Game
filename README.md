## 💢Features
- #### Playing movie, music and sound effects
- #### A two player game simultaneously
- #### NO blinking in console, more satisfaction
- #### Unexpected Ui/Ux from a Console game :)
- #### Stroge player's data for sign in and history

## ⏬Usage and Installation
- ### Installing:
  - #### Download all the prepared game from [HERE](https://mega.nz/file/kvwVCILR#SD1rMvLfH7FbTkMOk7GAOyQqYiDQg4VOIf3yuHgcuB8).
  - #### Launch the .exe file.
> [!NOTE]
> It has to be fully downloaded. You must have VLC installed on your PC and Msys64 for better experience.

> [!TIP]
> The massive volume of this game, only comes from .wav files and included movies :)

- ### Using:
  - #### Download the main.c file. It helps you to change and modify the code, like changing the media player, addresses, etc.
  - #### Download all Media assest like sounds and movies from [HERE](https://mega.nz/file/R6pBCQxB#dtEZX1uuEvjsrPAW5ubZF3D-oa0ZIggzylSLvGmXV8g).
  - #### Modify the code as you want!
> [!TIP]
> Using JetBarin CLion is the greatest advice I can give you :))
